## Intended Use

Use these pages when you need to:

- translate dimensional outputs into operational actions
- prepare a pilot evidence pack before external assessment
- align board, risk, legal, and operations stakeholders on a common governance workflow
- ensure evidence records satisfy the EIS-01 four-component structure required for Tier 1 classification

The pilot pages are implementation guides. They complement the normative framework and certification documentation.
