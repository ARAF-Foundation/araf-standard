This section provides guidance on evidence requirements for autonomous system governance under the Agentic Risk Architecture Framework (ARAF).

## Evidence Standards

- [Evidence Standard](/evidence/evidence-standard/)
- [Evidence Infrastructure Standard (EIS-01)](/evidence/evidence-infrastructure-standard/)

## Key Concepts

- Authenticity, Integrity, Traceability, Exportability
- Four evidence categories: Design, Deployment, Operational, Outcome
- Evidence tiers and admissibility by certification tier
