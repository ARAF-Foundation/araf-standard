# GN-003: Governance Coherence Assessment

*Assessor Guidance Note GN-003*

This guidance note provides supplementary methodology for operational coherence evaluation. It is required reading for all assessments targeting ARAF Compliant or ARAF Certified determinations.

### Scope

GN-003 covers:

- the three coherence components: Authority Adherence, Control Exercise, Drift Detection
- evidence sampling procedures for operational coherence evaluation
- the 180-day assessment window in practice, including window commencement and what constitutes the assessment window start date
- handling organisations that have not been operating for 180 days
- GCI calculation and dimensional coherence profiles
- separation of design adequacy findings from operational coherence findings in the assessment report

### Relationship to Core Standard Documents

GN-003 is a supplementary assessor reference. It supports, but does not replace, normative requirements in:

- [Governance Coherence Addendum](/certification/governance-coherence-addendum/)
- [Certification Framework](/certification/certification-framework/)
- [Governance Benchmark Index](/certification/governance-benchmark-index/)
