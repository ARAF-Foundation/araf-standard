# GN-001: Conducting an ARAF Governance Assessment

*Assessor Guidance Note GN-001*

This guidance note provides end-to-end methodology for assessment practice, from engagement setup through report delivery. It is the primary operational reference for accredited assessors conducting ARAF governance assessments.

### Scope

GN-001 covers:

- engagement setup and pre-engagement independence checks
- system scoping and Decision Supply Chain mapping
- evidence collection procedures by category
- dimensional scoring guidance for D1 through D6
- multiplier assessment and compound weakness identification
- Governance Benchmark Index calculation
- coherence evaluation and GCI procedures
- assessment report drafting
- certification determination and communication
- post-assessment obligations

### Relationship to Other Documents

GN-001 is the foundational assessor methodology document. GN-002 (Cross-Border Deployment) and GN-003 (Governance Coherence Assessment) are supplementary documents that extend GN-001 for specific assessment contexts. All guidance notes must be read in conjunction with the [Assessor Accreditation Standard](/assessor-guidance/assessor-accreditation-standard/).
