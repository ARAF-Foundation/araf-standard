This page defines the canonical ontology used across ARAF specification, assessment, certification, and evidence artifacts.

## Scope

The governance ontology standardizes terminology and relationships for:

- decision actors and authority holders
- system and model components
- governance controls and control ownership
- evidence objects and chain-of-custody links
- assessment objects and scoring outputs
- certification statuses and lifecycle states

## Core Entity Classes

- Decision Subject
- Decision System
- Control Owner
- Evidence Artifact
- Assessment Record
- Certification Record

## Relationship Model

This section will define normative relationships and mandatory fields used for interoperability between assessors, evidence platforms, and institutional reporting systems.

Ontology changes will be versioned and cross-referenced with the ARAF version history.
