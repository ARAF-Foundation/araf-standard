# Agentic Risk Architecture Framework (ARAF)

**Version:** 3.0  
**Published:** 2026  
**Status:** Public open standard  
**License:** Creative Commons Attribution 4.0  
**Maintained by:** ARAF Standards Authority Ltd (a company limited by guarantee)

The Agentic Risk Architecture Framework (ARAF) is an open governance standard defining the classification and certification infrastructure required for autonomous system governance.
